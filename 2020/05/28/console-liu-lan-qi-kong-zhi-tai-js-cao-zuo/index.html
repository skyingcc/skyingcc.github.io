无标签
发布日期:   2020-05-28

进入console：

直接进入：按下Option + Command + J（Mac）或者Ctrl + Shift + J（Windows / Linux）

进入控制台以后，就可以在提示符后输入代码，然后按Enter键，代码就会执行。如果按Shift + Enter键，就是代码换行，不会触发执行

显示信息的命令
console.log(“log”); //用于普通信息输出
console.info(“info”);//用于提示性信息
console.error(“error”);//用于输出错误信息
console.log(“warn”);//用于输出提示信息
console.dirxml(“输出代码带标签”)；//
